Sliders Demo - Slideable & Dialable

Demo Information

Behaviors demonstrated:

  • Slideable: Linear sliders for value control
  • Dialable: Rotational dials/knobs for value input

Try this:

  • Horizontal Sliders: Drag the handle left/right to change values
  • Vertical Sliders: Drag the handle up/down to change values
  • Dials: Drag around the center point to rotate and change values
  • All controls have different ranges and step sizes
  • Watch the values update in real-time at the top of the canvas
