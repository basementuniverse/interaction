Select Demo - Multi-select, Single-select & Drag-to-select

Hold Ctrl while clicking or drag-selecting to add to the current selection.

Demo Information

Behaviors demonstrated:

  • Draggable: Drag any card around the board
  • Selectable: Click to select and click empty space to deselect
  • Multi-selectable: Most cards allow additive selection with Ctrl
  • Selection box: When enabled, dragging on empty space selects all touched multi-select items

Try this:

  • Click a card to select it
  • Hold Ctrl and click multiple Multi cards to build up a selection
  • Drag on empty canvas space to create a marquee selection box
  • Hold Ctrl while marquee selecting to add to the current selection
  • Click empty space without dragging to clear all selected items
  • Try the Single cards: they stay single-select only and ignore marquee selection
